.- Crear la bases de datos dellstore
	CREATE DATABASE dellstore2
	psql -U postgres -c "CREATE DATABASE dellstore2;" // en contenedor

.- Procesar el guión DDL de estructura
  psql -f 3_0_dellstore2_fisico_estructura.sql dellstore2

.- Procesar el guión DML de datros
  psql -f 3_0_dellstore2_fisico_datos.sql dellstore2
