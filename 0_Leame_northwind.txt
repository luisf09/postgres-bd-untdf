Instalacion:
------------
.- Crear la bases de datos northwind
	CREATE DATABASE northwind;
	#psql -U postgres -c "CREATE DATABASE northwind;" // en contenedor
.- Procesar el guión DDL de estructura
  psql -f 3_northwind_fisico_estructura.sql northwind
  psql -U postgres -f 3_northwind_fisico_estructura.sql northwind
  
.- Procesar el guión DML de datros
  psql -f 3_northwind_fisico_datos.sql northwind
  psql -U postgres -f 3_northwind_fisico_datos.sql northwind
